# **App Name**: DevOps Portfolio

## Core Features:

- Terminal Intro: Home page with a terminal-style intro displaying my name, title, and a brief summary of my experience using typewriter animation.
- Skills Showcase: Skills page showcasing my expertise in Linux, Docker, Terraform, CI/CD, and AWS with progress bars representing my proficiency level in each skill.
- GitHub Projects: Projects page integrating with GitHub to display my projects with CI/CD pipelines, including a link to the repository and a brief description of each project.
- Interactive Resume: Resume page with a downloadable PDF and an interactive CLI-style resume, where users can type commands to view different sections of my resume.
- Contact Terminal: Contact Me page with a terminal-style contact form or a `curl`-style interaction for sending messages.
- AI Resource Recommendations: Implement a generative AI powered tool that creates personalized recommendations for DevOps-related resources, such as articles, tutorials, or tools based on the user's skill set and projects.

## Style Guidelines:

- Primary color: Vibrant green (#98FF98) reminiscent of classic terminal displays, signifying growth and technology.
- Background color: Dark charcoal (#222222) emulating a terminal's backdrop for focus and contrast.
- Accent color: Electric blue (#7DF9FF) to highlight interactive elements, echoing digital interfaces and user engagement.
- Body and headline font: 'Space Grotesk' (sans-serif) for a modern, techy look that works well for headlines and short body text.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets.
- Use simple, line-based icons in a neon green color to represent different skills and technologies, maintaining a consistent terminal-inspired aesthetic.
- Subtle typewriter effect for text, command prompts, and other interactive elements to enhance the terminal experience.
